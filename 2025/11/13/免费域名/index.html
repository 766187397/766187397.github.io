免费域名

预算不够买 .com？
今天教你 0 元拿下支持 Cloudflare 的顶级级域名——.dpdns.org，无需信用卡、无需实名、只需 GitHub 账号，真正的“点击即拥有”！

🎁 什么是 .dpdns.org？

  • 公共后缀列表（PSL）成员 → 浏览器、证书商、Cloudflare 都把它当“正规军”
  • 完全免费 → 首年 365 天，到期前 180 天内一键续期，不限次数
  • 功能齐全 → 支持 NS、A、CNAME、MX、TXT，可托管到 Cloudflare，开 CDN、SSL、Workers 全安排
  • 前缀随意 → 只要没人抢，你的 name.dpdns.org 就是你的

🛠️ 申请 7 步走（跟着做 ≈ 3 分钟）

步骤 操作提示
① 准备账号 1 个 GitHub 账号（无门槛注册）
1 个常用邮箱（Gmail / QQ 都 OK）
② 注册平台账户 打开 dash.domain.digitalplat.org/auth/register
用户名 + 随机英文姓名 + 邮箱 + 随机地址（带 2 个英文逗号 格式）
③ 邮箱验证 收件箱点激活链接（若没收到翻垃圾邮箱）
④ 登录 + KYC 首次登录会提示 GitHub OAuth → 点 “Authorize EdwardLab” 即可秒过
⑤ 抢域名 进入 域名注册页
输入想要的名称 → 选 .dpdns.org → Check → 填 2 个 NS（可先乱填）→ Register!
登录进来后首页往下滚动一点就有，标题是 “🎁 +1 Extra Domain Slot for Those Who Starred Us!”
⑥ 加号秘籍 想要 再薅 1 个？给官方仓库 DigitalPlatDev/FreeDomain 点 Star → 访问 验证页 点亮 +1 额度
⑦ 托管 Cloudflare CF 里 “添加站点” → 抄下分配的 NS → 回到 DP 后台修改 NS → 3 分钟生效，SSL、CDN 全开

📌 注意事项（避免翻车）

  1. 注册信息别太离谱——地址一定要带 2 个英文逗号，例：123 Main St, San Francisco, CA
  2. 域名一年有效期，记得 到期前 180 天内手动续期（免费但需主动点按钮）
  3. 禁止滥用——发垃圾邮件、钓鱼、挖矿会被秒封且无法申诉
  4. 平台非营利，如出现短暂维护，耐心等待即可
