OpenClaw安装与使用指南

什么是OpenClaw？

OpenClaw 是一款开源的个人 AI 智能体（Personal AI Agent），核心优势为本地部署 + 系统级执行 + 全渠道交互。它可以通过聊天工具（Telegram、WhatsApp、Discord等）下达指令，自动完成文件整理、代码部署、日程同步等任务，所有数据本地存储，隐私零泄露。

核心特性

  • 本地部署：数据不出本地，隐私安全有保障
  • 系统级执行：可执行 Shell 命令、访问文件系统
  • 多渠道接入：支持 Telegram、WhatsApp、Discord、飞书、钉钉等
  • 多模型支持：支持 OpenAI、Anthropic、智谱 GLM、阿里云百炼等多种 AI 模型

接入聊天渠道

接入 Telegram

步骤：

  1. 在 Telegram 中搜索 @BotFather
  2. 发送 /newbot 创建机器人
  3. 按提示输入机器人名称和用户名（需以 bot 结尾）
  4. 获取 Bot Token（格式：123456:ABC-DEF1234ghIkl）

接入 Discord

步骤：

  1. 访问 Discord 开发者门户：https://discord.com/developers/applications
  2. 创建新应用，进入 Bot 标签页
  3. 开启 Bot 并获取 Bot Token
  4. 在 OAuth2 -> URL Generator 中勾选 bot 权限
  5. 生成邀请链接，将 Bot 邀请到服务器

接入飞书

步骤：

  1. 访问飞书开放平台：https://open.feishu.cn/
  2. 创建企业自建应用
  3. 在凭证与基础信息中获取 App ID 和 App Secret
  4. 开启机器人权限

接入钉钉

步骤：

  1. 访问钉钉开放平台：https://open.dingtalk.com/
  2. 创建企业内部应用
  3. 获取 AppKey 和 AppSecret

安全注意事项

⚠️ 重要提示：OpenClaw 具备系统级权限，可以执行 Shell 命令和访问文件系统。

推荐做法

  1. 不要在主力电脑上运行：建议使用闲置设备或云服务器
  2. 限制权限：仅启用必要的权限（文件系统访问、Shell访问）
  3. 禁止网络访问权限：防止恶意代码外泄
  4. 定期审查日志：检查异常操作
  5. 谨慎安装第三方 Skills：从可信来源安装，检查代码

Windows 用户特别提示

Windows 用户建议使用 WSL2（Linux 子系统）运行，因为 Windows 的文件系统权限模型较为脆弱。

Skills 安全警告

安装第三方 Skills 时要特别注意：

  • 检查 Skills 的来源和代码
  • 不要安装来源不明的 Skills
  • 定期审查已安装的 Skills
