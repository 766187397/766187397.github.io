  2. 怎么使元素上下左右居中? (Flex知识)

  3. 单位px、em、rem的区别?

    • px是固定不变的像素单位
    • em是继承父元素字体大小的一种相对单位
    • rem是继承根元素字体大小的相对单位

  4. 如何实现移动端页面适配?

    • rem + 媒体查询
    • JS插件（amfe-flexible + postcss-pxtorem）
      • amfe-flexible动态设置 REM 基准值（html 标签的字体大小）
      • postcss-pxtorem将 px 转为 rem
    • 小程序使用rpx

JavaScript

  1. js如何删除数组的第n项?

    • 使用splice(开始下标，结束下标，添加)
    • 使用delete arr[n]长度不变，原位置为undefined

  2. js如何判断数组中各项[是否有]或[是否都是]包含指定字符串?

    • indexOf方法返回在数组中可以找到给定元素的第一个索引，如果不存在，则返回 -1。
    • includes方法用来判断一个数组是否包含一个指定的值，根据情况，如果包含则返回 true，否则返回 false。
    • every方法测试一个数组内的所有元素是否都能通过某个指定函数的测试。它返回一个布尔值。
    • some方法测试数组中是不是至少有 1 个元素通过了被提供的函数测试。它返回的是一个 Boolean 类型的值。

  3. 阐述一下 js中的原型链的感念?

    • Object都有一个特殊的属性prototype，也就是原形，当调用方法时Object自身没有就会通过原形一层层往上找，就是原形链，最终找不到就返回undefined
    • 原型链使得实例对象可以共享构造函数的原型属性和方法，节省内存

  4. js中this指向问题， 以及如何修改this指向?

    • 全局：window
    • 对象中的函数：对象
    • 事件绑定中：节点
    • 箭头函数：上层的this
    • 构造函数：实例
    • 改变this：call、apply和bind
      • call：xxx.call(yyy,参数1,参数2…)；立即执行，返回结果
      • apply：xxx.apply(yyy,数组对象)；立即执行，返回结果
      • bind：xxx.apply(yyy,数组对象)；不会立即执行，返回函数

Vue

  1. Vue中， 如何修改数组的某一项的值?

    • 使用Vue包装的方法
    • Vue.set：修改
      • 参数：Vue.set(对象/数组，下标，修改/添加的值)
    • Vue.delete：删除
      • 参数：Vue.delete(对象/数组，下标)

  2. Vue父 子组件之间传值都有哪些方法?并具体说明如何相互传值。

    • 父子

      • props：父传子
      • $emit：子传父

    • vuex

      • this.$store.state.数据属性
      • …mapState([‘数据属性1’,’数据属性2’])

    • 路由

      • query：{path:””,query:{xxx:’abc’,xxx:123}}
      • params：{name:””,params:{xxx:’abc’,xxx:123}}

    • 插槽（slot）

      • 默认插槽：直接在组件之间写
      • 具名插槽：<template v-slot:name>
      • 作用域插槽：子组件在slot上面写自定义属性=”数据”，父组件在调用子组件中间写template标签，使用slot-scope=”自定义属性”

    • ref

      • 可以获取组件的属性/方法

    • 数据总线

  3. 您 在工作中封装过哪些组件?如何进行Vue组件封装?

    • 案例（后台管理）：
      • 对表单进行封装，在登录和各个模块的编辑部分大量使用：
        • 通过表单type的特性，使用v-for，v-if等属性进行封装，使用时传递对应的规则即可生成对应的表单，并且内置插槽
      • 对按钮就行封装，由于涉及权限，对按钮封装：不可见，不可编辑，可编辑三种不同的格式
      • 对echarts进行封装，预留了一些必要的配置项
      • 对菜单列表进行封装，对后端传递的不同权限的用户展示不同的菜单选项

  5. Vue中 如何实现权限控制?

    前端权限大致分为：接⼝权限、按钮权限，页⾯权限，路由权限

    • 接⼝权限：头部携带token就⾏了（axios请求拦截器实现）
    • 按钮权限：v-if、v-show、自定义指令、disabled等
    • 菜单权限：获取菜单权限列表，动态递归⽣成菜单
    • 路由权限：路由导航守卫 + route.addRoute()

微信小程序

  1. 请讲一下微信授权的过程

    1. 第一步：用户同意授权，获取code
    2. 第二步：通过 code 换取网页授权access_token
    3. 第三步：刷新access_token（如果需要）
    4. 第四步：拉取用户信息(需 scope 为 snsapi_userinfo)
    5. 附：检验授权凭证（access_token）是否有效

  2. 请讲一下小程序 登录的过程

    1. wx.login() 获取 临时登录凭证 code
    2. 通过code换取网页授权access_token（后端获取OpenID）
    3. 拉取用户信息

  3. openid和unionid的区别?

    • openid：微信小程序/微信公众号，用户的唯一标识
    • unionid：不同应用间识别用户的唯一性

综合能力

  1. 什么是跨域?如何解决?

    浏览器的同源策略造成的，是一种安全策略

    协议、域名、端口就会产生跨域

    解决方法

    • JSONP：只能获取Get请求，还需要服务器支持
    • CORS：资源共享，需要后端开启
    • 代理：中间件代理->相当于服务器访问服务器

  2. 从页面打开到显示都发生了什么?

    1. 首先在浏览器中输入URL
    2. 查找缓存
    3. DNS域名解析
    4. 建立TCP连接
    5. 发起HTTP请求：三次握手
    6. 服务器响应请求并返回结果
    7. 关闭TCP连接
    8. 浏览器渲染
    9. JS引擎解析过程
